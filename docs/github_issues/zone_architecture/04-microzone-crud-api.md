## Summary
Implement Microzone CRUD API endpoints nested under zones.

## Scope
- Add microzone to zone
- Rename microzone
- Delete microzone
- Toggle microzone visibility
- List microzones per zone

## Acceptance Criteria
- All microzone endpoints are scoped to a parent zone.
- Ownership and auth are enforced.
- Area is computed and returned in km2.
- API supports efficient retrieval for zone card expansion.
